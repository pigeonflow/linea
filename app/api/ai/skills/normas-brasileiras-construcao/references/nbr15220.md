## 1. Zoneamento Bioclimático Brasileiro

O Brasil é dividido em **8 zonas bioclimáticas** com base em dados climáticos (temperatura, umidade, radiação). Cada zona possui diretrizes construtivas específicas.

### 1.1 Resumo das Zonas

| Zona | Clima Predominante | Cidades Representativas |
|---|---|---|
| Z1 | Frio (inverno rigoroso) | Campos do Jordão (SP), Urubici (SC), São Joaquim (SC) |
| Z2 | Frio a temperado | Curitiba (PR), Lages (SC), Caxias do Sul (RS) |
| Z3 | Temperado | Porto Alegre (RS), São Paulo (SP), Belo Horizonte (MG) |
| Z4 | Temperado a quente | Brasília (DF), Franca (SP), Florianópolis (SC) |
| Z5 | Quente (litoral úmido) | Vitória (ES), Niterói (RJ), Santos (SP) |
| Z6 | Quente semiárido | Goiânia (GO), Campo Grande (MS), Montes Claros (MG) |
| Z7 | Quente seco (semiárido intenso) | Petrolina (PE), Juazeiro (BA), Caicó (RN) |
| Z8 | Quente úmido (equatorial) | Manaus (AM), Belém (PA), Macapá (AP), Recife (PE), Salvador (BA), Rio de Janeiro (RJ) |

> **Nota importante:** A classificação de cada município está no Anexo A da NBR 15220-3. A lista acima é apenas ilustrativa. Muitos municípios têm classificação diferente do esperado. Sempre consultar o anexo oficial.

|---|---|---|---|---|---|
| Z1 | Pesadas e isoladas | Leve isolada | Mínima no inverno | Forte | Parcial no verão |

### 2.2 Detalhamento por Zona

#### Zonas 1 e 2 (Frio)

| Parâmetro | Requisito |
|---|---|
| Parede externa | Pesada: U ≤ 3,0 W/(m²·K), atraso térmico φ ≥ 6,5 h |
| Cobertura | Leve isolada: U ≤ 2,0 W/(m²·K), atraso térmico φ ≥ 3,3 h |
| Aberturas para ventilação | Pequenas: 10% a 15% da área de piso |
| Sombreamento de aberturas | Permitir entrada de sol no inverno |
| Estratégia principal | Aquecimento solar passivo, isolamento térmico |

#### Zonas 3 e 4 (Temperado)

| Parâmetro | Requisito |
|---|---|
| Parede externa | Pesada: U ≤ 3,7 W/(m²·K), atraso térmico φ ≥ 4,3 h |
| Cobertura | Leve isolada: U ≤ 2,0 W/(m²·K), atraso térmico φ ≥ 3,3 h |
| Aberturas para ventilação | Médias: 15% a 25% da área de piso |
| Sombreamento de aberturas | Sombrear no verão, permitir sol no inverno |
| Estratégia principal | Inércia térmica, ventilação cruzada |

#### Zonas 5 e 6 (Quente)

| Parâmetro | Requisito |
|---|---|
| Parede externa | Pesada: U ≤ 3,7 W/(m²·K), atraso térmico φ ≥ 6,5 h |
| Cobertura | Leve isolada: U ≤ 2,0 W/(m²·K) |
| Aberturas para ventilação | Z5: Grandes (≥ 40% da área de piso, ventilação cruzada); Z6: Médias (15% a 25%, ventilação seletiva) |
| Sombreamento de aberturas | Sombrear o ano todo |
| Estratégia principal | Ventilação cruzada (Z5), resfriamento evaporativo e ventilação seletiva (Z6) |

#### Zona 7 (Quente seco / semiárido)

| Parâmetro | Requisito |
|---|---|
| Parede externa | Pesada: U ≤ 2,2 W/(m²·K), atraso térmico φ ≥ 6,5 h |
| Cobertura | Pesada: U ≤ 2,0 W/(m²·K), atraso térmico φ ≥ 6,5 h |
| Aberturas para ventilação | Pequenas: 10% a 15% da área de piso (ventilação seletiva) |
| Sombreamento de aberturas | Sombrear o ano todo |
| Estratégia principal | Inércia térmica pesada, resfriamento evaporativo, ventilação seletiva (abrir à noite) |

#### Zona 8 (Quente úmido / equatorial)

| Parâmetro | Requisito |
|---|---|
| Parede externa | Leve refletora: U ≤ 3,6 W/(m²·K), atraso térmico φ ≤ 4,3 h |
| Cobertura | Leve refletora: U ≤ 2,3 FV W/(m²·K), atraso térmico φ ≤ 3,3 h |
| Aberturas para ventilação | Grandes: ≥ 40% da área de piso (ventilação cruzada permanente) |
| Sombreamento de aberturas | Sombrear o ano todo |
| Estratégia principal | Ventilação cruzada permanente, vedações leves, cores claras |

## 7. Fórmulas de Cálculo

### 7.1 Resistência Térmica Total (RT)

```
RT = Rsi + R1 + R2 + ... + Rn + Rar + Rse
```

Onde:
- Rsi = resistência superficial interna
- Rse = resistência superficial externa
- Rn = espessura(n) / λ(n) [resistência de cada camada]
- Rar = resistência da câmara de ar (se houver)

### 7.2 Transmitância Térmica (U)

```
U = 1 / RT   [W/(m²·K)]
```

### 7.3 Capacidade Térmica (CT)

```
CT = Σ (λᵢ × ρᵢ × cᵢ × eᵢ) / λᵢ = Σ (ρᵢ × cᵢ × eᵢ)   [kJ/(m²·K)]
```

Onde:
- ρ = densidade do material [kg/m³]
- c = calor específico [kJ/(kg·K)]
- e = espessura da camada [m]

### 7.4 Atraso Térmico (φ)

O atraso térmico é calculado conforme procedimento da NBR 15220-2 e depende da composição das camadas. Valores típicos são dados nas tabelas de componentes.
